# Search Lab — руководство пользователя

## Что это

Search Lab — это платформа для **сравнения алгоритмов поиска** на **OpenSearch**: вы настраиваете режимы поиска, ведёте **библиотеку запросов** с разметкой релевантности и запускаете **матрицы бенчмарков** (N алгоритмов × M запросов) с едиными метриками качества. Результаты сохраняются для сравнения во времени. Это инструмент **оценки и экспериментов**, а не замена production-поиску.

## Кому подойдёт

Исследователям, поисковым инженерам и всем, кому нужно на одних и тех же данных сравнивать **BM25**, **семантический (векторный)**, **гибридный** и **RRF** подходы.
